按下面做即可完成上传到 GitHub
========================================

一、在「终端」里执行（复制整段粘贴）：

cd /Users/bssm/katherine-zhang-website

# 若之前初始化失败，先删掉不完整的 .git 再初始化（二选一）
rm -rf .git
git init
git branch -M main

git add .
git commit -m "Initial commit: Katherine Zhang personal website"


二、在浏览器里：

1. 打开 https://github.com/new
2. Repository name 填：katherine-zhang-website
3. 选 Public，不要勾选 Add a README
4. 点 Create repository


三、回到终端，执行（把 YOUR_USERNAME 换成你的 GitHub 用户名）：

git remote add origin https://github.com/YOUR_USERNAME/katherine-zhang-website.git
git push -u origin main


按提示登录或授权后，就上传完成了。

若已添加过 origin，只需执行：git push -u origin main
