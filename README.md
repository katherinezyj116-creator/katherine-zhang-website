# Katherine Zhang — Personal Website

Two-page personal site: **About** (name + bio) and **Portfolio**.

## Run locally

Open `index.html` in a browser, or use a local server:

```bash
cd katherine-zhang-website
python3 -m http.server 8000
```

Then visit: http://localhost:8000

## Edit

- **index.html** — Name and bio text
- **portfolio.html** — Portfolio items (titles, descriptions, add/remove cards)
- **styles.css** — Colors, fonts, spacing

Replace the placeholder portfolio thumbnails with real images by using `<img>` inside each `.thumb` div and removing the placeholder text.
